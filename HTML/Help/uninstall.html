  • Using JavaNews

Uninstalling JavaNews is quite easy. As it is designed for multiple platforms there aren't any entries in the registry or control panel uninstall list. To ensure that the application is removed fully from your computer you must ensure that the following items are deleted:

  • Shortcuts to the application
  • The application installation directory
  • The database used by JavaNews
  • Any log files (*.log.xml) created by the application
  • DSN entries (if using ODBC) relating to the JavaNews database

Using the automated uninstaller

If you used the automated installer there will be a shortcut for the uninstaller in the start-menu. You can use this to uninstall most aspects of the application. Items that aren't removed using this include:

  • The ODBC configuration
  • Any log files
  • The database (if you're not using the original file)
