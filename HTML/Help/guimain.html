The main application window is the parent window of all other windows used by JavaNews. While using the application it is the outer-most border and menu.

Contained within the menus are a number of options.

The File menu

The only option currently available in this menu is to exit the application.

The Configuration menu

The contents of the configuration menu is quite extensive and has a dedicated section that explains all of its features. It can be found here.

The Window menu

This contains two options allowing you to either minimise all of the windows open in JavaNews or rearrange and tile them. Minimised windows appear as small bars along the bottom edge of the window. Tiling windows will resize and rearrange them so that all windows are the same size and arranged to fit in the whole of the visible space.

The Help menu

The help menu contains information that may be of use to you. There is more information available here.
