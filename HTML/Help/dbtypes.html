  • Using JavaNews

JavaNews has been built to use Java DataBase Connectivity (JDBC) and through this it is able to use a variety of database types. Any database which is ODBC compliant can be used to store the back-end data required for the operation of the JavaNews application.

Other types of database may be supported but you should refer to further documentation which can be found here.

If you are using MacOS, Linux, Unix or another operating system then use of an emulator maybe required to obtain ODBC functionality. The application has not been tested outside of a Microsoft Windows environment.
