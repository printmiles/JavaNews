This section contains technical information regarding the structure and operation of the application. This is aimed at people who are concerned with the operation of the application such as programmers, database administrators and advanced users.

Javadoc

Javadoc is added by the developer during coding the functions and features of the application. The tags used in Javadoc are compiled into a set of HTML files that can be viewed to document the content and function of the application. This can be found here.

Entity Relationship (E-R) Diagram

An entity relationship diagram describes the structure of a database. Columns, tables and keys within the database make its structure and this is what an E-R diagram attempts to document. An E-R diagram for JavaNews can be viewed here.

UML Diagrams

Unified Modelling Language is used to depict how programs work, what their functions are and the manner in which messages can be passed between classes. UML diagrams for JavaNews can be found here.

For more information about using and reading UML diagrams, the Wikipedia entry is a good first point of reference and can be found here.
