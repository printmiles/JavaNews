Unified Modelling Language (UML) diagrams have been provided that document the structure of the application, known as class diagrams. There are also diagrams that show how classes communication with one another, known as sequence diagrams. Finally there is a diagram that shows how the application responds to data input from the attached COM ports, known as an activity diagram.

All of the diagrams listed below are in Adobe portable document format (PDF) files and require Adobe Acrobat reader v5.0 or higher to read them. If you do not have this application then please see below for more information.
