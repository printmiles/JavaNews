In addition to the main help files, these appendices have been included to assist you with the workings of JavaNews should you need to look "under the bonnet" of the application.

Preference Files

These files are used by the application to store settings about how it should look, the level of logging that should be used and information about the COM ports it should use.

Log Files

These are used by the application to store information about actions taken within the application and details about any errors that may have occurred. This can be used for auditing purposes and for help debugging errors.

JNFixer

This is a helper application designed to perform two functions: verification of database structure and batch import of records into the database.
