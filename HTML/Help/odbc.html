ODBC stands for Open DataBase Connectivity and is used in some operating systems to provide a common means to access a variety of database types.

Editing data source names (DSNs) can be performed through the ODBC panel. In Windows XP and Vista this can be found in the Administrative Tools panel. You can usually access this either through the Start Menu or through the Control Panel. You may need administrative rights to access the control panel and the ODBC page.

The process of configuring and editing DSNs is shown below as a step-by-step guide.

Configuring & Editing DSNs

Once you have located the ODBC data source panel, double click on the shortcut and you will be presented with a list of different types of DSNs (User, System and File) and tabs to install other ODBC drivers and perform some configuration. These are not important to us and you should make sure that the User DSN tab is highlighted. This will show a listing of those DSNs you have configured, as well as those configured by the system on your behalf.

If you have already configured a link to the JavaNews database then it will be listed here. You can click on its name in the list and "Configure" to change its settings.

If you have not configured a DSN to the JavaNews database then click on "Add".
  • The ODBC configuration window. (Click to enlarge)

You will see a list of the different drivers already installed on your machine. These maybe in different languages but find the driver you require in the list in your native language.

There is a blank copy of a Microsoft Access database included in the JavaNews directory. If you wish to use this, select "Microsoft Access Driver (*.mdb)" from the list.

Click on Finish when you are ready to proceed.
  • Available ODBC drivers. (Click to enlarge)

If you are using the blank database included in the JavaNews installation folder then you will be presented with the Microsoft Access setup window (shown). Otherwise you will be presented with a similar window for your chose database type.
  • Entering database details. (Click to enlarge)

In most cases you will need to provide a name and description of the database you are connecting to. The name you give here is the one which needs to be entered into JavaNews as shown later.

Once you have done this, and you are using the supplied Microsoft Access database then click on "Select..."
  • Enter a name and description. (Click to enlarge)

You will need to locate the JavaNews.mdb file which is located in the JavaNews installation directory. Select the database file and click on OK to proceed.
  • Select the database file. (Click to enlarge)

Ensure that all of the information is correct before clicking on OK to save the DSN and add it to the list of configured data sources.
  • Check the information before saving. (Click to enlarge)

Once configured correctly the DSN will be listed in the Data Sources window. You can now use this database with JavaNews.
  • New DSN in the ODBC window. (Click to enlarge)
