Java applications can be awkward to install on a target PC, but a number of technologies are able to assist users when it comes to installing and uninstalling them.

Many of the supporting files required for JavaNews are contained within a central file. This is the JavaNews.jar file that is required for the application to work. However, due to the amount of additional configuration that is required for JavaNews to work, it is difficult to write an installer that can create all of the necessary configuration for the application to work as it was designed.

There is a lot of information here and most of it is to walk through some configuration aspects which you might not be familiar with. If you are new to Java and configuring your PC environment then I would advise that you look at the topics in the following order:

  1. Supported operating systems
  2. ODBC - Creating & editing DSNs
  3. Installing Java
  4. Installing the application
  5. Creating shortcuts
  6. Uninstalling JavaNews

There is an automated installation available for the Windows platform which allows for items 4-6 to be performed with a single file. For more about this please refer to the section "installing the application".

Check-list before starting JavaNews for the first time

To ensure that JavaNews will work the first time, you must ensure that you have:

  1. Installed a version of Java 1.6 or higher
  2. Configured a database to store records from JavaNews (there is a blank .MDB file included in the installation CD that can be copied and renamed to hold data)
  3. Configured an ODBC data source to the database you're using in step 2 (above).
