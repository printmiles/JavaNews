Other database types are not currently supported other than ODBC. If you do need to use a particular type of database then it is easiest and quicker to locate the ODBC drivers for that database type. To configure an ODBC DSN you should refer to the documentation here.

If you need to build a new back-end database then make sure that it emulates the structure of the current database. You can see the structure of the database as it is documented through an Entity-Relationship diagram here.
