Java allows for the look and feel (or theme) of an application written in it to be changed. The normal Java runtime allows for various operating systems to be catered for as well as Java's cross-platform Metal theme. Using Java 1.6 or higher allows for the Nimbus theme to be used as well.

Changing the theme in JavaNews, or any other Java application, does not affect the application in anyway and is purely aesthetic. You can choose from any of the following look and feel:
