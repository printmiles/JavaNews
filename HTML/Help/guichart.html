The screen shown below is the same for both recognised (known) links to the database and new (discovered) links while the application is running. The window incorporates a number of features which are discussed below.

The charts only record those events from the moment the window is opened.

The chart on the right-hand side (showing historical data) will be blank until a link changes state. This is because the chart only has one point to start with an requires a second data point to draw a line between them. Points are added when a LINK UP or LINK DOWN message is received.

The image shows two charts one pie chart, showing the current number of links and their states, and a line chart, showing past changes in the values of link states.

You are able to hove the mouse over items shown in either chart to display more detail. The above screenshot shows the tool-tip when the mouse hovers over the Link Up section of the pie chart. The detail shows the name of the item you're hovering on, the actual value and the percentage of all values it composes.

The line chart operates in a similar way with the bottom axis representing time. This allows for the time and date of peaks and troughs to be shown in the tool-tip when the mouse is hovered over a data point on the chart. Further options are displayed if you right-click on either of the graphs. These options allow you to alter the axes of the chart, the title and print the chart itself or save it as a Portable Network Graphics (PNG) file.
