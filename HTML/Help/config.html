The configuration menu in JavaNews allows for various aspects of the application to be customised. Any changes that are made are saved within the preferences for future use.

Most of the settings require the application to be closed and restarted before they take effect. This is because the objects they affect are used while the application is open. However, those that don't require the application to be restarted are:
