How to identify the problem

There are no links shown either in the reports section on the left-hand side of the monitoring window or on the right-hand side of the window.

What causes the problem

Failure to access the database either because the location of the database has changed or the DSN name has been altered. This problem can also be caused if there are no link records in the database at all.

How to fix it

Refer to the ODBC configuration to ensure that the database is configured and accessible. The ODBC configuration guide can be found here.

If there are no records in the database then you can you JNFixer to import some records or manually enter them into the database directly.
