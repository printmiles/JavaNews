When running the application for the first time a few items need to be configured so that running it on subsequent occasions, errors are not generated. These items that should be configured need to be done so in the following order and are:

  1. The logging level. See here for more information.
  2. The logging directory. This defaults to C:\JavaNews\Logs. If you need to change this then please refer to the instructions here.
  3. The database name and type need to be set. See here for more information.
  4. The COM port(s) to be monitored must be selected and configured. See here for more information.

Once all of these items have been configured then JavaNews is ready to go! All of these items are saved in the preferences for further use and this doesn't need to be changed again unless needed.
